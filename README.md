## 🚀 Tecnologias utilizadas

- ReactJS
- Styled Components
- Java Script

## 📥 Executar esse projeto no seu computador

- Clonar Repositório: `git clone <URL do projeto>`
- Instalar dependências: `yarn install`
- Rodar Aplicação: `yarn start`
